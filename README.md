# BSG-PabloPatch (Tabletop Simulator) 🎭

Ein grafischer Mod-Patch für die **Twilight Imperium IV** Implementation im Tabletop Simulator. Dieser Patch ersetzt diverse Texturen und Charakter-Gesichter durch das "Pablo"-Design.

## ✨ Features
* **Custom TTS Textures:** Speziell optimierte Bilddateien für den Import in TTS.
* **Pablo-fied Characters:** Ersetzt die Standard-Artworks der Charakterbögen und Karten durch Pablo-Varianten.
* **Einfache Installation:** Automatischer Windows-Installer oder universelles Archiv für alle Plattformen.

## 🚀 Installation

Gehe zu den **[Releases](https://github.com/Oxx0r/TI4-PabloPatch/releases)** und lade die für dich passende Datei herunter:

### 🛠️ Option A: Windows (Empfohlen)
1. Lade die Datei `TI4_Pablo_Patch_Installer_vX.X.X.exe` herunter.
2. Starte den Installer. Dieser findet deinen TTS-Ordner automatisch unter:
   `Documents\My Games\Tabletop Simulator\Mods`
3. Folge den Anweisungen und starte Tabletop Simulator neu.

### 📦 Option B: Manuelle Installation (Linux / Mac)
1. Lade die Datei `TI4_Pablo_Patch_vX.X.X.zip` herunter.
2. Entpacke den Inhalt direkt in dein Tabletop Simulator Mods Verzeichnis.
   - **Standard-Pfad:** `Documents/My Games/Tabletop Simulator/Mods/`
3. Bestätige das Überschreiben vorhandener Dateien, falls du gefragt wirst.
